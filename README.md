# Conversor de Moedas

Aplicação web simples que converte valores entre diferentes moedas, desenvolvida utilizando **HTML, CSS e JavaScript**.

## 📷 Preview do Projeto

![Preview do Conversor](conversor-preview.png)

## 💱 Funcionalidades

* Conversão entre diferentes moedas
* Interface simples e fácil de utilizar
* Cálculo dinâmico utilizando JavaScript
* Opção para inverter as moedas selecionadas

## 🛠 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
